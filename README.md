# Zelda with Java2d

RPG game like Zelda using Java2d.


## Prerequires

1. Git 2.6+
2. Maven 3.3+
3. Java 8+


## How to Play

Clone

```
git clone https://github.com/humbertodias/game-zelda-java2d.git
```

Inside the folder

```
cd game-zelda-java2d
```

Run

```
mvn package exec:java -Dexec.mainClass="zelda.java2d.Main"
```
